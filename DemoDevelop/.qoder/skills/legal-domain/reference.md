# 法律领域知识参考

## 1. 合同类型分类

### 商业合同
| 英文 | 中文 | 常见场景 |
|------|------|---------|
| Non-Disclosure Agreement (NDA) | 保密协议 | 商业合作前签署 |
| Service Agreement | 服务协议 | B2B 服务外包 |
| Employment Agreement | 雇佣合同 | 员工入职 |
| Shareholders' Agreement | 股东协议 | 公司治理 |
| License Agreement | 许可协议 | 知识产权授权 |
| Distribution Agreement | 分销协议 | 产品分销渠道 |
| Lease Agreement | 租赁合同 | 办公场地租赁 |
| Sales and Purchase Agreement (SPA) | 买卖合同 | 资产或股权交易 |
| Joint Venture Agreement | 合资协议 | 合资企业设立 |
| Franchise Agreement | 特许经营协议 | 品牌加盟 |

### 合同关键条款
| 英文 | 中文 | 审查要点 |
|------|------|---------|
| Term and Termination | 期限与终止 | 起止日期、自动续期、提前终止条件 |
| Indemnification | 赔偿条款 | 赔偿范围、上限、免责事由 |
| Limitation of Liability | 责任限制 | 损害赔偿上限、排除间接损失 |
| Governing Law | 适用法律 | 管辖地法律选择 |
| Dispute Resolution | 争议解决 | 仲裁 vs 诉讼、管辖法院 |
| Force Majeure | 不可抗力 | 免责事件定义和通知义务 |
| Confidentiality | 保密义务 | 保密信息范围、期限 |
| Intellectual Property | 知识产权 | 归属、许可、转让 |

## 2. 新加坡法律合规要求

### 数据保护 - PDPA (Personal Data Protection Act)
- **收集限制**: 仅收集合理必要的个人数据
- **使用限制**: 仅用于收集时告知的用途
- **同意要求**: 收集/使用/披露前需获得同意
- **数据准确性**: 确保个人数据准确完整
- **保留限制**: 不再需要时应删除或匿名化
- **跨境传输**: 跨境转移需确保接收方有同等保护
- **数据泄露通知**: 重大数据泄露须 3 天内通知 PDPC

### 电子交易 - ETA (Electronic Transactions Act)
- 电子签名在新加坡具有法律效力
- 电子合同与纸质合同同等有效
- 某些例外（遗嘱、房产转让等）不适用

### GST (Goods and Services Tax)
- 标准税率: 9%（2024年起）
- B2B 服务: 适用标准税率
- 跨境数字服务: 海外供应商需注册 GST（如年营业额超 S$100万）
- Invoice 必须包含: 供应商 GST 注册号、税额、总额

## 3. 法律 AI 交互模式

### 合同审核 (Contract Review)
```
用户上传合同 → AI 识别合同类型 → 逐条款分析 →
标记风险点（高/中/低） → 生成审核报告 →
提供修改建议 → 用户确认
```

关键输出要素:
- 风险等级标记（红/黄/绿）
- 问题条款原文引用
- 修改建议和替代措辞
- 行业惯例参考

### 法律研究 (Legal Research)
```
用户输入法律问题 → AI 理解问题域 →
检索相关法规和判例 → 分析适用性 →
产出研究报告 → 附引用来源
```

### 法律翻译 (Legal Translation)
```
用户上传文档 → AI 识别语言对（中→英 / 英→中） →
逐段翻译 → 法律术语标准化 →
专业术语高亮 → 用户审校
```

翻译注意事项:
- 法律术语必须使用标准译法（参考上方术语表）
- 保留原文的条款编号结构
- 日期和金额格式转换
- 法律管辖地名称使用官方译名

## 4. 法律文书常见结构

### 标准合同结构
1. **标题** (Title)
2. **当事方** (Parties)
3. **鉴于条款** (Recitals / Whereas)
4. **定义与解释** (Definitions and Interpretation)
5. **主体条款** (Operative Provisions)
6. **陈述与保证** (Representations and Warranties)
7. **赔偿** (Indemnification)
8. **责任限制** (Limitation of Liability)
9. **保密** (Confidentiality)
10. **期限与终止** (Term and Termination)
11. **一般条款** (General Provisions)
12. **适用法律与争议解决** (Governing Law and Dispute Resolution)
13. **签署页** (Execution Page)
14. **附件** (Schedules/Annexures)
